# Aqbobek Artificial Assistant - Project Features

## Обзор проекта

Aqbobek Artificial Assistant - это комплексная система управления школой с интеграцией ИИ для автоматизации рутинных задач директора и учителей.

## Основные компоненты

### 📁 Backend (`/backend`)
- **API сервер** (FastAPI) - HTTP API для взаимодействия с дашбордом
- **Telegram бот** - для взаимодействия сотрудников через мессенджер
- **Обработка голоса** - транскрибация через Groq Whisper
- **NLP обработка** - классификация сообщений и извлечение данных
- **База данных** - работа с Supabase
- **Планировщик** - автоматические задачи (ежедневные отчёты)

### 📁 Frontend (`/frontend`)
- **Веб-дашборд** (Next.js) - панель управления для директора
- **Распознавание речи** - Web Speech API для голосовых команд
- **Интерфейс управления** - задачи, расписание, сотрудники, инциденты
- **RAG система** - генерация официальных документов на базе приказов

### 📁 Database (`/supabase`)
- **PostgreSQL** - основная база данных
- **Realtime** - уведомления в реальном времени
- **Migrations** - управление схемой базы данных

## Ключевые функции

### 🎯 Управление задачами
- Создание задач голосом или текстом
- Назначение исполнителей
- Отслеживание сроков выполнения
- Уведомления в Telegram

### 🍽️ Столовая и посещаемость
- Отчёты учителей по посещаемости
- Автоматический подсчёт порций
- Ежедневные отчёты директору
- Баллы за своевременные отчёты

### 📅 Расписание уроков
- Генерация расписания с помощью CP-SAT
- Drag & Drop редактирование
- Валидация замен
- Общий график сотрудников

### 🔄 Замены учителей
- Автоматический поиск кандидатов
- Push-уведомления в Telegram
- Подтверждение/отклонение замен
- Интеграция с расписанием

### 📄 Документы (RAG)
- Генерация официальных распоряжений
- База знаний приказов
- Упрощение сложных пунктов
- Поиск по документам

### 🤖 Telegram бот
- Команды для учителей
- Отчёты по столовой
- Голосовые сообщения
- Управление заменами

### 🔔 Уведомления
- Telegram уведомления
- Push-уведомления
- WhatsApp интеграция (опционально)

## Технологический стек

### Backend
- Python 3.11+
- FastAPI
- Supabase (PostgreSQL)
- Groq API (Whisper, Llama)
- OpenAI API
- Aiogram (Telegram bot)
- APScheduler (планировщик)

### Frontend
- Next.js 14 (App Router)
- TypeScript
- Supabase
- TailwindCSS
- Lucide Icons
- shadcn/ui

### Database
- PostgreSQL (Supabase)
- pgvector (для векторного поиска)
- Realtime subscriptions
